Introdução

Sobre mim:

Desde 2018, quando comecei a cursar Engenharia de Software na UFC, estudo e desenvolvo aplicações para pôr o que aprendi em prática e obter experiência a partir disso sempre priorizando otimização, segurança e eficiência de forma mais rápida para cada sistema desenvolvido assim como seguindo os padrões de codificação de cada linguagem. Desde 2020 trabalho com frontend tendo desenvolvidos projetos utilizando javascript e suas funcionalidade extras como ajax, html, css e bootstrap.

Projetos Desenvolvidos

App de randomização para a plataforma android desenvolvido usando a linguagem java com o android studio.

Sistema web para gerenciamento de estoque utilizando no frontend javascript, html e css.

Sistema web para gerenciamento de horas complementares para alunos, com back-end desenvolvido em PHP inserido no Heroku e MongoDB atlas como o banco de dados, e o frontend desenvolvido utlizando javascript + react native. Na comunicação do frontend com o backend foi utilizado o ajax do PHP.

Sistema web de player de música utilizando php para buscar os arquivos na memória, javascript para controlar as funcionalidades na página, ajax para se comunicar com o código php e html e css para o design da página.

Assistente virtual desenvolvida em python utilizando algoritmos de deep learning treinados do 0, possuindo uma rede neural CNN para reconhecimento de gestos e outra para conversação.

Sistema para criação de notas desenvolvido em C# e utilizando como banco de dados o MySql.

Jogo de batalha naval utilizando algoritmos de machine learning treinados do zero a partir de um dataset criado, utlizando 2 modelos para comparção de desempenho entre ambos um deles sendo utilizado o algoritmo KNN e o outro Naive Bayes.

Algoritmo DPLL desenvolvido em python para solução de fórmulas lógicas no padrão FNC.

Simples interface para visualização da previsão do clima desenvolvido em C# e utilizando web scrapper para coletar os dados.
